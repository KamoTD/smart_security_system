SYSTEM OVERVIEW

A Smart Security System is a first line of defence for keeping your property and valuables safe from intruders. The need of the system is to help companies, businesses, but more especially home properties to be more secured and to protect expensive equipment and save lives 24/7. The system consists of two parts which is the hardware and software application. Both applications interact with each other to ensure security, the designed hardware can be placed in a place where you would like the intruder to be detected and where you can arm and disarm the alarm and the software(Web Application) will be used for remote access and able control the system remotely
